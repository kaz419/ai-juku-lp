# 中小企業向けAI活用オンライン塾 事業企画書

**作成日:** 2026年2月19日  
**企画者:** Kazuki Miyazaki  
**事業名:** AI活用実践塾（仮称）  
**運営体制:** 人間2名 + AIチーム（Mike + Nexus - OpenClaw）

---

## 1. 背景・課題認識

### 1.1 市場環境

- **AI技術の急速な普及**: ChatGPT、Gemini、Claude等の生成AIが一般化し、業務効率化のポテンシャルは飛躍的に向上
- **中小企業の出遅れ**: 大企業はAI専門部署を設置し活用が進む一方、中小企業は「何から始めればいいか分からない」状態
- **情報の質の問題**: 書籍やYouTubeの情報は表面的で、実務に直結する実践ノウハウが不足

### 1.2 中小企業が抱える課題

| 課題 | 具体的な状況 |
|-----|------------|
| **知識不足** | AIツールの存在は知っているが、自社業務への適用方法が分からない |
| **人材不足** | IT人材がおらず、社内に教えられる人がいない |
| **予算制約** | 外部コンサルを雇う数百万円の予算がない |
| **時間不足** | 経営者・社員とも忙しく、独学する時間がない |
| **情報の陳腐化** | 書籍を買っても、出版時点で情報が古くなっている |

### 1.3 既存サービスの問題点

- **一般向けAI講座**: 個人向けで、法人の実務に即していない
- **高額コンサル**: 数百万〜数千万円で中小企業には手が出ない
- **録画講座**: 一方通行で質問できず、最新情報へのアップデートがない
- **技術解説重視**: 「ChatGPTとは何か」ではなく「明日から何ができるか」が知りたい

## 2. 事業目的

### 2.1 ミッション

**中小企業が「明日から使える」AI活用スキルを習得し、大企業に負けない競争力を手に入れる**

### 2.2 ビジョン

- **短期（1年）**: 日本全国100社の中小企業がAIで業務効率化を実現
- **中期（3年）**: 1,000社に拡大し、業界別ベストプラクティスを確立
- **長期（5年）**: 中小企業のデジタル変革（DX）のデファクトスタンダードに

### 2.3 提供価値

1. **即効性**: 受講翌日から実務で使えるスキル
2. **実践性**: デモではなく、自分の業務で実際に動かす
3. **最新性**: 毎週アップデートされる最新情報
4. **継続性**: 単発ではなく、常に進化し続ける学習環境
5. **コミュニティ**: 同じ課題を持つ仲間との情報交換

## 4. サービス概要

### 4.1 サービス名

**「AI活用実践塾」**（仮称）

### 4.2 サービス内容

| 項目 | 内容 |
|-----|------|
| **形式** | 週1回2時間のオンラインライブ講義 + 録画アーカイブ |
| **スタイル** | 講師がデモ実演 → 受講者も同時に操作して体得 |
| **プラットフォーム** | Zoom（ライブ） + Slack/Discord（質問・交流） |

| **サポート** | **Mike（AIアシスタント）が24時間質問対応** + 人間による週1回Q&A |
| **アーカイブ** | 過去の講義動画を無制限視聴可（**Nexusが自動編集**） |
| **運営体制** | **人間2名 + OpenClaw AIチーム（超低コスト運営）** |

### 4.3 差別化ポイント

| 一般的なAI講座 | 本サービス |
|--------------|-----------|
| 個人向け | **法人向け（何名でもOK）** |
| 座学・理論中心 | **実践・デモ中心** |
| 録画のみ | **ライブ + Q&A** |
| 汎用的内容 | **業界別カスタマイズ** |
| 単発講座 | **毎週アップデート** |
| 質問できない | **24時間チャット + 週1人間対応** |

## 【Month 1】スマホでできるAI活用入門

### Week 1: 今日から使えるスマホAI（2時間）

**目標:** 受講翌日からスマホAIで業務効率化

**前半60分: デモ実演**
1. **Gmailの返信を30秒で自動生成**
   - ツール: Gemini for Gmail（無料）
   - デモ: 「お見積もり依頼メール」への返信を自動生成
   - Before: 5分かけて返信文作成 → After: 30秒で下書き完成
   - プロンプト例: 「丁寧かつ簡潔に、見積もり送付の旨を伝える返信文を作成」

2. **音声入力で議事録を5分で作成**
   - ツール: Google音声入力 + ChatGPT（無料）
   - デモ: 10分間の録音 → 議事録に整形
   - Before: 30分かけて手打ち → After: 音声入力2分 + AI整形3分
   - プロンプト例: 「以下の音声入力テキストを議事録形式に整理してください」

3. **スマホでPDF資料を要約**
   - ツール: NotebookLM（無料）
   - デモ: 20ページの契約書PDFを3行で要約
   - プロンプト例: 「この契約書の重要ポイントを3点に絞って箇条書きで」

**中盤30分: 受講者も一緒に操作**
- ステップ1: Gemini for Gmailをインストール（講師が画面共有で誘導）
- ステップ2: 自分の過去メールで返信文を自動生成してみる
- ステップ3: 音声入力で簡単な文章を作成 → ChatGPTで整形
- ステップ4: 自社の資料PDFをNotebookLMにアップロードして要約

**後半30分: Q&A + 宿題**
- よくある質問:
  - 「有料版と無料版の違いは?」→ 無料版で十分使える、有料は速度と精度が向上
  - 「セキュリティは大丈夫?」→ 社外秘情報の扱い方を解説
- 宿題: 今週のメール返信を全てAIで下書き作成してみる

**受講生が持ち帰れる成果物:**
- Gmailの自動返信テンプレート3種類
- 議事録自動生成のプロンプト集
- PDF要約のチェックリスト

### Week 2: 会議を10倍効率化（2時間）

**目標:** 会議録音から議事録・タスク・フォローアップメールまで自動生成

**前半60分: デモ実演**
1. **録音から議事録を自動生成（Otter.ai）**
   - ツール: Otter.ai（無料版300分/月）
   - デモ: 30分会議の録音 → 5分で議事録完成
   - 使い方:
     - スマホでOtter.aiアプリ起動 → 録音ボタン押すだけ
     - 自動で話者認識（「発言者A」「発言者B」）
     - 終了後、テキストがメール送信される
   - Before: 1時間かけて議事録作成 → After: 5分で9割完成

2. **議事録からタスクと期日を自動抽出（ChatGPT）**
   - デモ: Otter.aiの議事録 → ChatGPTにコピペ
   - プロンプト例:
     ```
     以下の議事録から、
     - 誰が
     - 何を
     - いつまでに
     やるべきかをリスト化してください。
     ```
   - Before: 議事録を読み返して手動でタスク抽出 → After: 30秒で完了

3. **フォローアップメールを自動作成**
   - プロンプト例:
     ```
     この議事録をもとに、参加者全員に送る
     「本日の会議のまとめとNext Action」メールを作成してください。
     件名も考えてください。
     ```
   - デモ: Before/Afterを比較（5分で完成）

**中盤30分: 受講者も一緒に操作**
- ステップ1: Otter.aiアカウント作成（講師が画面共有）
- ステップ2: 録音機能のテスト（受講者同士で2分会話して録音）
- ステップ3: 録音テキストをChatGPTでタスク抽出
- ステップ4: フォローアップメールのドラフト作成

**後半30分: Q&A + 宿題**
- よくある質問:
  - 「日本語にも対応してる?」→ 対応済み、精度も高い
  - 「録音は相手の許可が必要?」→ 会議冒頭で一言伝えるのがベスト
- 宿題: 次回の社内会議をOtter.aiで録音 → 議事録作成

**受講生が持ち帰れる成果物:**
- Otter.ai設定マニュアル（PDF）
- タスク抽出用プロンプト集
- フォローアップメールのテンプレート3種類

## 【Month 2】営業・提案業務の効率化

### Week 5: 商談録音から提案書を自動生成（2時間）★超重要★

**目標:** 商談30分後には提案書ドラフトが完成している状態を実現

**前半60分: デモ実演（完全実演）**

**STEP 1: 商談を録音（Otter.ai / tldv）**
- ツール: 
  - 対面商談 → Otter.ai（スマホで録音）
  - Zoom商談 → tldv（自動録画・文字起こし）
- デモ:
  - 架空の商談（「新規顧客からのホームページ制作依頼」）を実演
  - 録音開始 → 5分間の商談デモ → 録音終了
  - 自動で文字起こし完了（所要時間1分）

**STEP 2: 文字起こしテキストをChatGPTに投入**
- プロンプト（完全公開）:
  ```
  あなたはプロの営業です。以下の商談録音から、
  顧客が求めていることを抽出し、提案書のドラフトを作成してください。

  【必須項目】
  1. 顧客の課題・ニーズ
  2. 提案する解決策
  3. 提案内容の詳細（3〜5項目）
  4. 期待される効果
  5. 概算見積もり（価格帯）
  6. 納期の目安

  【商談録音】
  （ここにOtter.aiのテキストをコピペ）
  ```

- デモ結果（Before/After比較）:
  - **Before**: 商談後、2〜3時間かけて手動で提案書作成
  - **After**: 商談終了5分後にはドラフト完成

**STEP 3: AIが生成した提案書を人間が最終調整**
- AIのドラフトをベースに:
  - 価格を正確な数字に修正
  - 会社情報・担当者名を追加
  - デザイン調整（Googleドキュメント/Word）
- Before: ゼロから2時間 → After: 調整のみ30分

**STEP 4: 提案書をPDFで自動送信**
- ツール: Gmail + Gemini
- プロンプト:
  ```
  以下の提案書PDFを添付して、
  顧客に送る丁寧なメール文を作成してください。
  件名も考えてください。
  ```
- デモ: メール文も30秒で完成

**実際のデモ画面構成:**
- 左画面: Otter.aiの文字起こし結果
- 右画面: ChatGPTで提案書を生成中
- → 完成した提案書をGoogleドキュメントで表示
- → 最終調整版をPDFエクスポート
- → Gmailでメール送信

**中盤30分: 受講者も一緒に操作**
- ステップ1: 事前に用意した「サンプル商談録音テキスト」を配布
- ステップ2: 各自がChatGPTに投入して提案書作成
- ステップ3: できた提案書をSlackに投稿して見せ合う
- ステップ4: プロンプトの改善ポイントをディスカッション

**後半30分: 業界別カスタマイズ**
- 製造業向け: 「RFQ（見積依頼）から見積書を自動生成」
- サービス業向け: 「ヒアリングシートから企画書を自動生成」
- 不動産業向け: 「内見時の会話から物件提案書を自動生成」

**受講生が持ち帰れる成果物:**

**Q2: 何名まで参加できますか？**
A: 同一法人内であれば何名でも参加OKです。社長1名でも、営業部全員でも、料金は同じです。

A: はい、全講義の録画をアーカイブ化しており、無制限で視聴可能です。リアルタイムで参加できなくても安心です。

**Q5: 業種は問いませんか？**
A: 製造業、サービス業、小売業、建設業、不動産業など、業種を問わず活用いただけます。業界別の事例も毎月紹介しています。

**Q6: 有料版のAIツールを契約する必要がありますか？**
A: 初月は無料版のみで十分です。慣れてきたらChatGPT Plus（月20ドル）の契約を推奨しますが、必須ではありません。
