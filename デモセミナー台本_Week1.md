#### **STEP 1: 商談を録音（Otter.ai）**

**画面操作:**
1. Otter.aiアプリを開く（スマホ画面共有、またはPC版）
2. 「録音」ボタンを押す
3. 架空の商談を実演（5分）

**商談シナリオ例:**
```
【設定】
- 顧客: 株式会社ABC（製造業、従業員50名）
- 担当者: 田中部長
- 課題: ホームページが古くて採用に苦戦
- 予算: 100〜150万円
- 納期: 3ヶ月以内

【商談内容（講師が一人二役で実演）】

講師（営業）:
「田中部長、本日はお時間いただきありがとうございます。
 現在のホームページの課題をお聞かせください。」

講師（顧客役）:
「そうですね、今のホームページは10年前に作ったもので、
 デザインが古くて、スマホでも見にくいんです。
 特に採用ページが弱くて、応募者が全然来ないんですよ。」

講師（営業）:
「なるほど、採用強化が最優先なんですね。
 予算感と納期はいかがですか？」

講師（顧客役）:
「予算は100〜150万円くらいで考えています。
 納期は、できれば3ヶ月以内に完成させたいです。
 春の採用シーズンに間に合わせたいので。」

講師（営業）:
「承知しました。では、
 採用ページを中心に、スマホ対応のデザインで
 リニューアルのご提案をさせていただきます。
 後ほど提案書をお送りしますね。」

講師（顧客役）:
「よろしくお願いします！」
```

**トーク（録音終了後）:**
```
はい、これで商談終了です。
録音を止めます。

Otter.aiが自動で文字起こしを開始します。
（画面を見せながら）
ほら、もう文字起こしが始まっていますね。

これが約1分で完了します。
```
