## 🌟 キラーコンテンツ: 提案書自動生成の実演

### 📌 Week 5の詳細（受講生から最も評価が高い回）

**Before（従来の提案書作成）:**
```
商談終了 → 記憶を頼りに2時間かけて提案書作成
→ 記載漏れ、顧客の要望を正確に反映できないリスク
```

**After（AI活用）:**
```
商談録音（Otter.ai / tldv）
   ↓
文字起こし（自動、1分で完了）
   ↓
ChatGPTに投入（専用プロンプト使用）
   ↓
提案書ドラフト完成（5分）
   ↓
人間が微調整（価格・デザイン等、30分）
   ↓
完成！（合計35分、従来比75%削減）
```

### 🛠️ 使用ツール

| ツール | 用途 | 料金 |
|--------|------|------|
| **Otter.ai** | 商談録音→文字起こし | 無料版OK（月300分） |
| **ChatGPT** | 提案書ドラフト生成 | 無料版OK（Plus推奨） |
| **Googleドキュメント** | 最終調整・PDF化 | 無料 |

### 📝 実際のプロンプト例（講義で公開）

```
あなたはプロの営業です。以下の商談録音から、
顧客が求めていることを抽出し、提案書のドラフトを作成してください。

【必須項目】
1. 顧客の課題・ニーズ
2. 提案する解決策
3. 提案内容の詳細（3〜5項目）
4. 期待される効果
5. 概算見積もり（価格帯）
6. 納期の目安

【商談録音】
（ここにOtter.aiのテキストをコピペ）
```

### 📊 実例: Before/After比較

**Before（手作業）:**
- 時間: 2時間
- 精度: 記憶頼り（漏れあり）
- ストレス: 高

**After（AI活用）:**
- 時間: 35分（75%削減）
- 精度: 録音ベースで正確
- ストレス: 低（微調整のみ）
